Quiero crear un proyecto en python
Que usando la libreria mouse, grabe la posicion de 9 clicks del raton y guarde las posiciones en un fichero, asociando cada posicion a numero, siendo el primer click un 1
y el ultimo un 9.
Que usando el fichero de posiciones y otro mas que contendra una lista de numero de 4 digitos (codes.txt) y la libreria mouse, haga clicks con el raton en las posiciones asociadas
a cada digito.
Que guarde en un fichero para tener memoria persistente el ultimo numero de lista usado.

Flujo

El usuario inicia el script
El script comprueba si el fichero de posiciones existe, si no ejecuta metodo de grabacion, al terminar los 9 clicks se guardan en un fichero.
Si el usuario pulsa la tecla F5 el metodo de grabacion volvera a ejecutarse.
Se inicia un metodo que ejecuta los clicks en pantalla segun lo explicado anteriormente. Cada numero (4 clicks) se ejecutara siguiendo el orden actual de la lista de codes.txt
El bind para que el script introduzca el siguiente numero es la tecla F6.

Por lo tanto hasta que el usuario no pulse la tecla F6 no se introducira el primer numero y el script quedara a la espera.
Una vez pulsada la tecla F6 se haran los 4 clicks y el script volvera a quedar a la espera.

Logea toda la informacion relevante y muestrala por consola.
